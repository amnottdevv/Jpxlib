# Installation

JPX can be installed by downloading the latest release from the repository.

Currently only **v1.0** is available.

## Download

1. Go to the **Releases** section of this repository.
2. Download the file from **JPX v1.0**.
3. Extract the archive.

## Run

After extracting, open a terminal in the extracted folder and run:

```
jpx yourfile.jpx
```

Example:

```
jpx example.jpx
```

## Verify Installation

Run:

```
jpx --version
```

Expected output:

```
JPX v1.0
```

## Notes

- Only the **v1.0 release** is currently available.
- Future versions may include additional installation methods.
